Case Study Expansion

Deep Dive: Building an AI-Powered Satellite Dashboard

Challenge

One of the most challenging projects I worked on was building an AI-powered satellite tracking dashboard. The main difficulty was handling real-time satellite data while keeping the application responsive and accurate.

Approach

On the backend, I implemented APIs to fetch and process TLE data and optimized performance using caching techniques.

I also integrated a local LLM to answer user queries, designing a grounded system so responses were based only on relevant satellite data.

Outcome

This experience improved my ability to design full-stack systems, optimize performance, and integrate technologies like React, Node.js, and APIs effectively.
